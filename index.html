Загальні Правила

Основні Правила

Правило 1

Адміністратори повинні бути ввічливими та професійними у спілкуванні з гравцями.

Правило 2

Усі рішення адміністраторів повинні бути обґрунтовані та справедливі.

Правило 3

Адміністратори повинні дотримуватися конфіденційності інформації про гравців.

Правила Викрадень

Основні Правила Викрадень

  1. Викрадення повинні бути обґрунтовані рольовою грою.
  2. Заборонено викрадати гравців у безпечних зонах.
  3. Викрадачі повинні мати чіткий план дій.
  4. Викрадення повинні бути узгоджені з адміністрацією.

Правила для Викрадачів

  1. Викрадачі повинні мати маски.
  2. Викрадачі повинні мати вагомі причини для викрадення.
  3. Викрадачі повинні дотримуватися встановлених сум викупу.

Правила Терактів

Основні Правила Терактів

  1. Теракти повинні бути узгоджені з адміністрацією.
  2. Теракти повинні мати чіткий план дій.
  3. Теракти повинні бути обґрунтовані рольовою грою.

Правила для Терактів

  1. Теракти повинні мати мінімум 10 учасників.
  2. Теракти повинні мати чіткі вимоги.
  3. Теракти повинні бути проведені у встановлений час.

Правила Постачання

Основні Правила Постачання

  1. Постачання повинні бути узгоджені з адміністрацією.
  2. Постачання повинні мати чіткий маршрут.
  3. Постачання повинні бути захищені.

Правила для Постачання

  1. Постачання повинні мати мінімум 5 охоронців.
  2. Постачання повинні бути проведені у встановлений час.
  3. Постачання повинні мати чіткі інструкції.

Війни Кланів

Основні Правила Війни Кланів

  1. Війни кланів повинні бути узгоджені з адміністрацією.
  2. Війни кланів повинні мати чіткі причини.
  3. Війни кланів повинні бути обґрунтовані рольовою грою.

Правила для Війни Кланів

  1. Війни кланів повинні мати мінімум 10 учасників з кожного боку.
  2. Війни кланів повинні мати чіткі умови перемоги.
  3. Війни кланів повинні бути проведені у встановлений час.
